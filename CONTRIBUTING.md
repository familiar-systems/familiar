# Contributing

This project isn't open to outside contributions yet. There's no triage process in place, and the codebase changes too fast for external PRs to land cleanly. This file will be updated when that changes.

For security reports, see [SECURITY.md](SECURITY.md). For everything else, feel free to open an issue to start a conversation; we may not act on it, but we'll see it.

If you've found a bug that affects you and you'd like a workaround, an issue is the right venue.
